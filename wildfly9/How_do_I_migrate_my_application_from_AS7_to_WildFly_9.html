How do I migrate my application from AS7 to WildFly 9

The purpose of this guide is to document changes that are needed to successfully run and deploy AS 7 applications on WildFly 9. It provides information on to resolve deployment and runtime problems and how to prevent changes in application behavior. This is the first step in moving to the new platform. Once the application is successfully deployed and running on the new platform, plans can be made to upgrade individual components to use the new functions and features of WildFly 9.
